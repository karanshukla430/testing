About This Project

This is an interactive web application with nested routing:

  • Home (/): Interactive form and dynamic content
  • Route A (/a): Main route with sub-navigation
  • Route A/B (/a/b): Sub-route within A
  • Route A/B/C (/a/b/c): Deepest nested route
  • Route B (/b): Data dashboard with live statistics
  • Route C (/c): Interactive tools

Navigate between routes using the navigation menus above!
